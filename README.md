# This is no-nude chat-roulette bot.

> [!IMPORTANT]
> **.env file requiered**
> example:
> ```.env
> export BOT_TOKEN=1234567890
> ```

# Installing

## Linux

```shell
git clone https://github.com/severnayaluna/nude_chat.git
cd nude_chat
python3 -m venv env
source env/bin/activate
pip install -r requirements.txt
python src/main.py
```

## Windows

```shell
git clone https://github.com/severnayaluna/nude_chat.git
cd nude_chat
python3 -m venv env
.\env\Scripts\activate
pip install -r requirements.txt
python src/main.py
```
