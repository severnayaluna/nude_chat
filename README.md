# This is no-nude chat-roulette bot.

> [!IMPORTANT]
> **.env file requiered!**
> 
> **~/.../nude_chat/.env**:
> ```
> export BOT_TOKEN=1234567890
> ```

# Installing

## Linux

```shell
git clone https://github.com/severnayaluna/nude_chat.git
cd nude_chat
python3 -m venv env
source env/bin/activate
pip install -r requirements.txt
```

## Windows

```shell
git clone https://github.com/severnayaluna/nude_chat.git
cd nude_chat
python3 -m venv env
.\env\Scripts\activate
pip install -r requirements.txt
```

# Running

## Linux

```shell
source env/bin/activate
python src/main.py
```

## Windows

```shell
.\env\Scripts\activate
python src/main.py
```
